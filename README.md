# Wikidata

Wikidata project for mapping black philly artists

## Overview

This is the institutional Temple University archive of SPARQL queries and analysis for Black artists in Philadelphia. This repository contains both the analytical notebooks and historical query result data.

## Files

- `Querying_and_Visualizing_Wikidata.ipynb` - Main analysis notebook with SPARQL queries and visualizations
- `querying_and_visualizing_wikidata.py` - Standalone Python script version (can be run from command line)
- `Wikidata Query 11_21_2020.csv` - Archived query results (Nov 21, 2020)
- `Wikidata Query 9_9_2021.csv` - Archived query results (Sept 9, 2021)

## Note

This is archived for institutional purposes. For the most up-to-date version with additional notebooks and tools, see the primary repository at: https://github.com/artvizphilly/wikidata
